# 📱 Projeto QR Code

Um projeto simples e funcional para geração de QR Codes desenvolvido com foco em praticidade e facilidade de uso.

## 🚀 Funcionalidades

- Gerar QR Code a partir de textos ou links
- Interface simples e intuitiva
- Geração instantânea
- Projeto leve e responsivo

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 📂 Estrutura do Projeto

```bash
Projeto-qrcode/
│
├── index.html
├── style.css
├── script.js
└── assets/
```

## ▶️ Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/amandaefgsantana-eng/Projeto-qrcode.git
```

2. Acesse a pasta do projeto:

```bash
cd Projeto-qrcode
```

3. Abra o arquivo `index.html` no navegador.

## 💻 Demonstração

Você pode executar o projeto localmente diretamente no navegador.

## 📸 Preview

```txt
Digite um texto ou link → Clique em gerar → QR Code criado ✅
```
